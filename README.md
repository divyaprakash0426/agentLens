# AgentLens

AgentLens is a framework-agnostic library to visualize AI-driven browser actions directly on a live page with an animated cursor, spotlight, and optional DOM execution.

## Packages

- `agentlens` (core)
- `@agentlens/parsers`
- `@agentlens/react`
- `@agentlens/vue`

## Development

```bash
pnpm install
pnpm typecheck
pnpm test
pnpm build
pnpm docs:dev
```
